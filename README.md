# Redis Practice

This is a Practical **Nextjs** project for practicing **Redis** with **Node.js**.

## Technologies

- Redis
- Nodejs
- Nextjs 13

## Features

- Connecting to **Redis**
- Adding only **unique** data
- Work with **Hashes**
- Work with **Sorted Sets**

## Installation

First add **.env.local file**. then:

```shell
npm install

node run dev
```
