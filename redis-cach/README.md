# Redis Cach

This is a Practical **Nodejs** project for practicing **Redis Cach**.

## Technologies

- Redis
- Nodejs

## Features

- Connecting to **Redis**
- Adding only **unique** data
- Work with **Hashes**
- Work with **Middlewares** & **Caching**
- Getting Data with **Node Fetch**

## Installation

First add **.env.local file**. then:

```shell
npm install

node run dev
```
